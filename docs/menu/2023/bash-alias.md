# git bash配置命令行别名技巧

## 配置

打开`git bash`，输入以下命令：

```shell
nano /etc/profile
```

光标下移动至最后，根据实际使用输入你想生成的命令行别名

![](/bash-alias/设置bash别名.png)

`alias` 后面接的是你想要设置的命令行别名，`=` 后面接的是你想要设置的命令行别名对应的命令。

## 使用
输入完成后，`ctrl + c`保存，重新启动`git bash`终端，直接输入别名即可使用。

![](/bash-alias/使用bash别名.png)

## 总结
实用性还是不错的，能减少输入的命令行长度，提高效率。
